# Relic Test Fixture Generator

This tool provides a PowerShell 7 script to generate large-scale, synthetic ROM libraries for performance, stress, and scalability testing of the Relic scanner and database index.

## Prerequisites

The generator requires **PowerShell 7** (`pwsh`). It will not run on legacy Windows PowerShell (powershell.exe).

## Parameters

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `-Root` | String | Yes | *None* | Absolute or relative path where the test library will be generated. |
| `-Systems` | String[] | No | `nes, snes, gb, gba, megadrive, psx` | Slugs of the systems to generate. Must match TOML slugs in `core/data/systems/*.toml`. |
| `-PerSystem` | Int | No | `100` | Number of game files to generate for each system. |
| `-WithMedia` | Switch | No | `False` | When specified, also emits cover art placeholders in a `media/covers/` folder. |
| `-Seed` | Int | No | `42` | Random seed to guarantee identical name and revision generation across runs. |
| `-Force` | Switch | No | `False` | Overwrite existing files. Running without `-Force` is idempotent and safe. |

## Usage Examples

### 1. Simple Smoke Test Library
Generate 100 game files for each of the 6 default systems (600 files total):
```powershell
pwsh -File tools/fixgen/generate.ps1 -Root C:\Users\username\relic-test-library
```

### 2. Large Performance Test Library (10k files)
Generate 1,666 game files for each of the 6 default systems (approx. 10,000 files total):
```powershell
pwsh -File tools/fixgen/generate.ps1 -Root C:\Users\username\relic-perf-library -PerSystem 1666
```

### 3. Large Performance Test Library with Cover Art
Include local artwork placeholders alongside the 10k ROM files:
```powershell
pwsh -File tools/fixgen/generate.ps1 -Root C:\Users\username\relic-perf-library -PerSystem 1666 -WithMedia
```

## Crucial Safety Warnings

> [!WARNING]
> **NEVER COMMIT GENERATED TREES**
> The files generated by this script are meant only for local performance diagnostics, profiling, and benchmarking. Do not commit these directories or files to the git repository.
